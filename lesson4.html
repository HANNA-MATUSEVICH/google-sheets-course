Тема 1: AI в Google Таблицах

Что доступно

  • Боковая панель Gemini — AI-ассистент
  • Автодополнение при вводе формул
  • Функция =AI() в некоторых планах Workspace
📌 Открыть: иконка Gemini в правом верхнем углу
⚠️ Если у вас нет кнопки «Ask Gemini» в таблице, возможно, ваш Workspace не разрешает использование AI из соображений конфиденциальности.

Что умеет Gemini

Задача Пример запроса
Создать таблицу «Создай таблицу для учёта расходов по категориям»
Написать формулу «Напиши формулу: сумма продаж только за март»

⚠️ AI — ускоритель, не замена пониманию.

AI для написания формул

Вместо запоминания синтакса — опишите задачу словами:

«Мне нужна формула, которая ищет имя сотрудника в столбце A, и если находит — возвращает его зарплату из столбца C, а если не находит — показывает "Не найдено"»
💡 Три сценария использования внешнего AI:
1. Написать формулу → скопировать → адаптировать ссылки
2. Очистить данные → вставить результат
3. Придумать структуру таблицы под задачу

Тема 2: Макросы

Макрос — записанная последовательность действий, которую можно запустить повторно.

💡 Если каждую пятницу вы делаете одно и то же —
запишите это один раз и запускайте кнопкой.

Два способа создания

📹 Запись макроса
Без кода, для начинающих
💻 Apps Script
JavaScript — для тех, кто хочет большего

Запись макроса — пошагово

  1. Расширения → Макросы → Записать макрос
  2. Выберите тип ссылок: абсолютные или относительные
  3. Выполняйте нужные действия в таблице
  4. Нажмите «Сохранить», дайте имя
  5. Готово — запуск через Расширения → Макросы → [имя]
💡 Можно назначить горячую клавишу: Ctrl+Alt+Shift+[цифра]

Apps Script — примеры кода

// Покрасить ячейку A1 в зелёный
function formatHeader() {
  var sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();

  var cell = sheet.getRange("A1");
  cell.setBackground("#0F9D58");
  cell.setFontColor("white");
  cell.setFontWeight("bold");
}

Кнопка на листе → запуск макроса

  1. Вставка → Рисунок → нарисуйте прямоугольник с текстом «Запустить»
  2. Вставьте рисунок
  3. Кликните на него → три точки → «Назначить скрипт»
  4. Введите имя функции
💡 Что можно автоматизировать:
форматирование отчётов · очистка данных · генерация шаблонов · отправка email

Триггеры — автозапуск

Скрипт можно запускать без нажатия кнопки.

Apps Script → Триггеры (иконка часов слева)

Событие Когда запускается
При открытии файла Каждый раз при открытии
При редактировании При любом изменении ячейки
По расписанию Каждый час / день / неделю

Практические задания

Задание 1 — Gemini

  1. Откройте боковую панель Gemini
  2. Попросите написать формулу для реальной задачи
  3. Примените и проверьте результат

Задание 2 — Макрос

  1. Сделайте текст в первой строке жирным + залейте ячейку синим + автоподбор ширины столбца
  2. Создайте кнопку на листе
  3. Привяжите макрос к кнопке
⭐ Необязательное задание: Напишите скрипт с помощью AI — опишите задачу в ChatGPT или Gemini, скопируйте код, запустите, объясните что делает каждая строка.
