Тема 1: Интерфейс Google Таблиц

Прежде чем работать с данными — нужно понять, где всё находится.

Это как изучить панель управления автомобиля перед первой поездкой.

🔑 5 ключевых элементов интерфейса

1. Строка названия файла

По умолчанию — «Untitled spreadsheet»

Кликните на название → переименуйте. Автосохранение работает всегда.

2. Строка меню

Выделены основные пункты: File / Edit / View / Insert / Format / Data

МенюКогда использовать
Файл (File)Создать копию, скачать, открыть доступ
Формат (Format)Оформление ячеек, условное форматирование
Данные (Data)Сортировка, фильтры
Вставка (Insert)Строки, столбцы, диаграммы

3. Панель инструментов

Быстрый доступ к самым частым действиям

💡 Меню — это «всё». Панель — это «самое нужное».

4. Рабочая область

Столбцы — буквы (A, B, C…) · Строки — числа (1, 2, 3…) · Ячейка = пересечение

📍 B3 — столбец B, строка 3. Это называется адрес ячейки.

5. Листы (вкладки внизу)

Один файл — много листов. Переименовывайте, копируйте, скрывайте.

💡 Используйте листы для разделения данных: «Данные», «Отчёт», «Справочник»

Тема 5: Условное форматирование

Таблица сама меняет цвет по заданному правилу — без ручной работы.

Путь: Format → Conditional formatting

Как настроить — пример «больше 10»

Условие «Greater than 10» → зелёная заливка. Ячейки 11, 45, 776 подсвечены автоматически.

Практические сценарии

СценарийУсловиеФормат
Статус задачиТекст = «Готово»🟢 Зелёный
ПросрочкаТекст = «Просрочено»🔴 Красный
ДоходЗначение > 0🟢 Зелёный
РасходЗначение < 0🔴 Красный
Дедлайн сегодняДата = СЕГОДНЯ()🟡 Жёлтый

Тепловая карта (Color Scale)

Цвет по градиенту — от минимального к максимальному значению.

Путь: Формат → Условное форматирование → Цветовая шкала

💡 Применение: анализ продаж по регионам, нагрузка по дням недели, оценки.
⚠️ Типичные ошибки:
• Правило применено не к тому диапазону
• Число хранится как текст → условие не срабатывает

Тема 6: Относительные и абсолютные ссылки

Одна из ключевых концепций. Без понимания ссылок — формулы не будут работать правильно.

Относительная (A1)
Сдвигается при копировании 
C1 = A1 + B1
C2 = A2 + B2  ← сдвиг
C3 = A3 + B3
Абсолютная ($A$1)
Не меняется при копировании 
= A1 * $B$1  ← $B$1 всегда
= A2 * $B$1  ← одна и та
= A3 * $B$1  ← же ячейка
💡 Клавиша F4 при редактировании ссылки — добавляет $ автоматически.

Таблица умножения — пример

ЗаписьЧто зафиксированоКогда использовать
$A$1Столбец и строкаСтавка, коэффициент — одна ячейка на весь расчёт
$A1Только столбецДанные из фиксированного столбца
A$1Только строкаДанные из фиксированной строки
Формула в B2 для таблицы умножения:
= $A2 * B$1

$A2 → столбец A фиксирован, строка сдвигается вниз
B$1 → строка 1 фиксирована, столбец сдвигается вправо

Тема 7: Даты и форматы чисел

Дата в Google Таблицах — это число.

Точка отсчёта: 30 декабря 1899 года

ДатаЧисло
01.01.202545658
02.01.202545659
💡 Именно поэтому с датами можно считать:
= B2 - A2 → количество дней между датами
= СЕГОДНЯ() - A2 → сколько дней прошло

Функции для работы с датами

ФункцияЧто делаетПример
СЕГОДНЯ()Текущая дата=СЕГОДНЯ()
ТДАТА()Дата и время=ТДАТА()
ДЕНЬ(A1)Число месяца→ 15
МЕСЯЦ(A1)Номер месяца→ 3
ГОД(A1)Год→ 2025
🌍 Формат отображения зависит от локали документа.
Изменить: Файл → Настройки → Регион
