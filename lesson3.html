Тема 1: Сводные таблицы

❌ Без сводной таблицы:
Писать СУММ.ЕСЛИ для каждой комбинации вручную
✅ Со сводной таблицей:
3 клика — и готово
📌 Сводная таблица = инструмент для быстрого анализа больших данных без написания формул.

Требования к исходным данным

  • ✅ Первая строка — заголовки столбцов (обязательно!)
  • ✅ В каждом столбце — один тип данных
  • ✅ Нет объединённых ячеек
  • ✅ Нет полностью пустых строк внутри данных
⚠️ Нарушение этих правил → сводная таблица будет работать некорректно.

Как создать сводную таблицу

  1. Кликните на любую ячейку внутри таблицы с данными
  2. Вставка → Сводная таблица
  3. Выберите «Новый лист» (рекомендуется)
  4. Откроется панель редактора справа

Четыре области редактора

Область Назначение Пример
Строки Группировка по строкам Менеджер
Столбцы Группировка по столбцам Квартал
Значения Что считаем СУММ продаж
Фильтры Фильтрация всей таблицы Регион
💡 Правый клик → Обновить — сводная не обновляется автоматически!

Функции агрегации

Функция Когда использовать
СУММ Деньги, количество
СЧЁТ Считает только ячейки с числами
СЧЁТЗ Считает все непустые ячейки, включая текст
СРЕДНЕЕ Средний чек, средняя оценка
МАХ / МИН Максимальная / минимальная сделка

Тема 2: Диаграммы

Число «578 000» — много или мало? Непонятно без контекста.

График мгновенно показывает тренды, сравнения и структуру.

Как создать

  1. Выделите данные (включая заголовки)
  2. Вставка → Диаграмма
  3. Таблица предложит тип автоматически

Какой тип выбрать?

Тип Когда Пример
📊 Столбчатая Сравнить категории Продажи по менеджерам
📈 Линейная Тренд во времени Продажи по месяцам
🥧 Круговая Доли (до 6 частей) Доля рынка
📉 Гистограмма Распределение Размер сделок
📊+📈 Комбо Два показателя Выручка + маржа%
⚠️ В визуализации данных круговая диаграмма с более чем 6 секторами крайне не рекомендуется: глазу трудно точно сравнивать углы секторов.

Настройка диаграммы

Вкладка «Настройка»

  • Тип диаграммы
  • Диапазон данных
  • Заголовки осей

Вкладка «Стиль»

  • Цвета рядов
  • Подписи данных
  • Положение легенды
💡 Чтобы перенести диаграмму на отдельный лист: нажмите три вертикальные точки в правом верхнем углу диаграммы → выберите «Переместить на отдельный лист».
Либо скопируйте на другой лист: Ctrl+C → Ctrl+V.

Практические задания

Задание 1 — Сводная таблица

  1. Создайте таблицу 20+ строк: Менеджер · Регион · Месяц · Сумма
  2. Сводная: строки — Менеджер, столбцы — Месяц, значения — СУММ
  3. Добавьте фильтр по Региону
  4. Сортируйте менеджеров по убыванию суммы

Задание 2 — Диаграммы

  1. Столбчатая: «Продажи по менеджерам»
  2. Линейная: «Динамика продаж по месяцам»
  3. Добавьте заголовки
  4. Включите подписи данных
