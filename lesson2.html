Тема 1: Фильтры

Фильтр временно скрывает строки, которые не подходят под условие.

📌 Данные не удаляются — просто не показываются.

Как включить

  1. Кликните на любую ячейку в таблице
  2. Данные → Создать фильтр
  3. В заголовках появятся кнопки ▼

Сохранить фильтр как шаблон

  1. Настройте нужный фильтр
  2. Данные → Режимы фильтрации → Сохранить как режим фильтра
  3. Дайте название: например «Только Минск»
💡 Сохранённый режим фильтрации не мешает другим пользователям.
Каждый включает свой фильтр независимо — это удобно при совместной работе.

Тема 2: Базовые формулы

⚠️ Любая формула начинается с =. Без него — это просто текст.
ФункцияЧто делаетПример
СУММСумма значений=СУММ(A1:A10)
СРЗНАЧСреднее значение=СРЗНАЧ(B2:B20)
СЧЁТКол-во ячеек с числами=СЧЁТ(C2:C100)
СЧЁТЗКол-во непустых ячеек=СЧЁТЗ(D2:D100)
МИН / МАКСМин / макс значение=МИН(A1:A50)

СУММ.ЕСЛИ — сумма по условию

= СУММ.ЕСЛИ(диапазон_условия; условие; диапазон_суммирования)

Пример: сумма продаж только по менеджеру «Иванов»

= СУММ.ЕСЛИ(A2:A100; "Иванов"; B2:B100)
СЧЁТЕСЛИ — считает строки по условию:
= СЧЁТЕСЛИ(A2:A100; "Иванов")
= СЧЁТЕСЛИ(B2:B100; ">1000")

Тема 3: Функция ЕСЛИ (IF)

Тема 4: ВПР (VLOOKUP)

Ищет значение в первом столбце таблицы и возвращает данные из другого столбца.

= ВПР(искомое_значение; таблица; номер_столбца; 0)
АргументЧто это
искомое_значениеЧто ищем (ячейка или текст)
таблицаДиапазон поиска (первый столбец = поиск)
номер_столбцаКакой столбец вернуть (1, 2, 3…)
0Точное совпадение (почти всегда нужно 0)

Пример ВПР

Справочник A:B
101 | Ноутбук
102 | Мышь
103 | Клавиатура

Формула — найти название по коду из D2:
= ВПР(D2; $A$1:$B$5; 2; 0)
         ↑            ↑
    что ищем      возвращаем 2-й столбец
⚠️ ВПР ищет только в первом столбце диапазона и возвращает только вправо.

XLOOKUP — современная замена ВПР

= XLOOKUP(что_ищем; диапазон_поиска; диапазон_результата; "Не найдено")
ВПРXLOOKUP
ПоискТолько в 1-м столбцеВ любом столбце
РезультатТолько правееВлево и вправо
Если не найдено#Н/ДСвой текст

Тема 6: Данные между листами и файлами

Ссылка на другой лист

= Лист2!A1                // значение с другого листа
= СУММ(Лист2!A1:A10)      // формула с другого листа
= 'Данные продаж'!B5      // если в названии есть пробел

Ссылка на другой файл — IMPORTRANGE

= IMPORTRANGE("https://docs.google.com/spreadsheets/d/XXX";
              "Лист1!A:D")
💡 IMPORTRANGE создаёт живую ссылку — данные обновляются автоматически.
При первом использовании нажмите «Разрешить доступ».
Если нужна статическая копия:
Ctrl+C → Ctrl+Shift+V → Только значения
